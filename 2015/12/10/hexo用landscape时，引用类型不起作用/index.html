hexo用landscape时，引用类型不起作用

今天在写笔记的时候，发现用macdown写出的笔记可以被正确识别。但是用hexo生成后，变成了居中的样式了。
我查看了我的样式与zipperary的样式的差别，发现了一些端倪。我的这里关于这个样式的定义如下:

.article-entry blockquote {
font-family: Georgia, “Times New Roman”, serif;
font-size: 1.4em;
margin: 1.6em 20px;
text-align: center;
}

这里的text-align是center，居中。而在zipperary的样式中，它引用到了另外一种截然不同的样式:

article .entry blockquote {
background: #f6f6f6;
border-left: 4px solid #999;
font-style: normal;
font-family: Georgia, ‘Times New Roman’, Times, STFangsong, KaiTi, serif;
font-size: 1.2em;
padding: 0 30px 15px;
padding-top: 10px;
padding-bottom: 30px;
}
