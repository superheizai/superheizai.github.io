回到顶部按钮

这两天想给这个在线笔记增加一个回到顶部的功能.

解决问题

在网上搜索了下，找到了两种基本的解决方案。典型例子的链接，在下面的参考资料中会给出. 我用第二个。为啥用第二个，是因为好看，方便还是啥原因？因为第一个不生效，然后用第二个还没生效，就在第二个上面解决问题，也就用了第二个。不过，第二个确实比第一个好看，还有个动态页面比例。
我这里要说的是，解决回到顶部按钮失效的问题。怎么破？给这个div加上z-index:9999999属性！

<style id='inline-css' type='text/css'>  
#backtoTop{
background-color:#eee;
border-radius:100%;
bottom:10%;height:48px;
position:fixed;
right:-100px;
width:48px;
transition:0.5s;
-webkit-transition:0.5s;
z-index: 9999999}  

#backtoTop.button--show{
right:10px}  

.per{
font-size:16px;
height:48px;
line-height:48px;
position:absolute;
text-align:center;
top:0;width:48px;
color:#555;cursor:pointer}

.per:before{
content:attr(data-percent)}  

.per:hover:before{
content:"↑";font-size:20px}  
</style>

尝试了两种方案，都没有起作用。都是看到了这个回到顶部的按钮，但是点击的时候没有任何反应。调试的时候，没有看到click事件发生。搞了两天，在和一位前端同学pair的时候，该同学发现右击审查元素的时候，并没有选中该元素，而是选中了该div背后的元素。于是乎，在chrome的elements中，找到该div元素，给他的对应css加上z-index属性，前端同学建议选择一个比较大的值，这里用了9999999. 设定好之后，it works!!!

具体做法

我用的第二种做法，基本可以分为两步，第一步是设定css文件，第二步是增加js文件。

设定css文件

如前两天的博文说的，我在写文档时候，用的“>“，引用标签，不起作用。我看了下，是因为生成的html中，对应的css样式不对，我从别人博客的样式中摘出来，覆盖到了自己的css文件，然后临时解决了问题。但是，这个只是临时解决方案，为什么呢？因为style.css是动态生成的，再次修改的话，之前修改的值还是会被覆盖掉。怎么破？
这里拿我们这次的backtotop的css文件举例。在themes/landscape/source/css下面，看到style.styl这个文件，它是集合所有css的入口。我们在合适的路径，比如_partial下面增加一个totop.styl，内容如上文所示，然后再style.styl顶部，加入@import “_partial/totop”.然后，重新generate一把，你就会发现你的新的css文件里面已经有了我们添加的样式。

增加js文件

在themes/landscape/source/js下面新增一个totop.js文件，然后在引入js地方，themes/landscape/layout/_partial/after_footer.ejs，最下方的<%- js(‘js/script’) %>之后，引入<%- js(‘js/totop’) %>，这样就完成了js的定义。

totop.js的内容如下: 

var bigfa_scroll = {
    drawCircle: function(id, percentage, color) {
        var width = jQuery(id).width();
        var height = jQuery(id).height();
        var radius = parseInt(width / 2.20);
        var position = width;
        var positionBy2 = position / 2;
        var bg = jQuery(id)[0];
        id = id.split("#");
        var ctx = bg.getContext("2d");
        var imd = null;
        var circ = Math.PI * 2;
        var quart = Math.PI / 2;
        ctx.clearRect(0, 0, width, height);
        ctx.beginPath();
        ctx.strokeStyle = color;
        ctx.lineCap = "square";
        ctx.closePath();
        ctx.fill();
        ctx.lineWidth = 3;
        imd = ctx.getImageData(0, 0, position, position);
        var draw = function(current, ctxPass) {
            ctxPass.putImageData(imd, 0, 0);
            ctxPass.beginPath();
            ctxPass.arc(positionBy2, positionBy2, radius, -(quart), ((circ) * current) - quart, false);
            ctxPass.stroke();
        }
        draw(percentage / 100, ctx);
    },
    backToTop: function($this) {
        $this.click(function() {
            jQuery("body,html").animate({
                scrollTop: 0
            },
            800);
            return false;
        });
    },
    scrollHook: function($this, color) {
        color = color ? color: "#000000";
        $this.scroll(function() {
            var docHeight = (jQuery(document).height() - jQuery(window).height()),
            $windowObj = $this,
            $per = jQuery(".per"),
            percentage = 0;
            defaultScroll = $windowObj.scrollTop();
            percentage = parseInt((defaultScroll / docHeight) * 100);
            var backToTop = jQuery("#backtoTop");
            if (backToTop.length > 0) {
                if ($windowObj.scrollTop() > 200) {
                    backToTop.addClass("button--show");
                } else {
                    backToTop.removeClass("button--show");
                }
                $per.attr("data-percent", percentage);
                bigfa_scroll.drawCircle("#backtoTopCanvas", percentage, color);
            }

        });
    }
}

jQuery(document).ready(function() {
    jQuery("body").append('<div id="backtoTop" data-action="gototop"><canvas id="backtoTopCanvas" width="48" height="48"></canvas><div class="per"></div></div>');
    var T = bigfa_scroll;
    T.backToTop(jQuery("#backtoTop"));
    T.scrollHook(jQuery(window), "#cc0000");
});
